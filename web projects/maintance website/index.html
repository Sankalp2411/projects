Rakshavahan

Armoured Defence Mobility Platform

Mission

Rakshavahan is an advanced, autonomous armoured vehicle platform designed for India's defence forces. Built for extreme terrain operations, it combines cutting-edge AI navigation, reinforced armour plating, and modular mission kits for surveillance, logistics, and tactical response.

Key Features

  • Autonomous terrain navigation with LiDAR + GPS fusion
  • Modular mission payloads — surveillance, supply, tactical
  • Reinforced composite armour rated for ballistic protection
  • Encrypted communication with real-time command relay
  • All-terrain mobility — desert, mountain, and amphibious

Tech Stack

AI / ML LiDAR Embedded Systems IoT Composite Materials

Lumiris

Intelligent Clean Energy System

Vision

Lumiris is an AI-powered clean energy management platform that optimizes solar panel arrays, battery storage systems, and micro-grid distribution. It makes renewable energy adoption seamless, efficient, and scalable for communities and industries alike.

Key Features

  • AI-driven solar tracking for maximum energy harvest
  • Smart grid management with real-time load balancing
  • Predictive maintenance alerts via IoT sensor network
  • Battery storage optimization with deep learning
  • Carbon footprint dashboard with actionable insights

Tech Stack

AI / ML Solar Tech IoT Smart Grid Data Analytics
