Cloudburst Prediction System

The Cloudburst Prediction System is a Python-based project designed to predict and visualize cloudburst risk across Indian cities using live weather data. The system integrates several key components: real-time weather data collection through the OpenWeatherMap API, preprocessing of raw data, feature engineering, machine learning-based prediction, and geospatial visualization. Its primary goal is to provide an interactive map highlighting high-risk areas for potential cloudburst events, helping authorities and citizens prepare for sudden rainfall and floods.

The system begins by fetching current weather data, including temperature, humidity, rainfall, wind speed, and cloud cover, for multiple Indian cities. This raw data is then cleaned and preprocessed to handle missing values and remove duplicates, making it ready for further analysis. Feature engineering is applied to select the most relevant meteorological parameters, while a logistic regression model is trained to predict the probability of cloudburst events. The model then classifies cities into high-risk or low-risk categories based on these predictions.

To make the predictions actionable, an interactive, India-wide map is generated using the Folium library. Each city is represented by a circular marker color-coded according to its cloudburst risk level, with red indicating high risk and green indicating low risk. Users can click on any marker to view detailed information such as temperature, humidity, rainfall, and predicted cloudburst risk for that city. The resulting map is browser-ready, portable, and easy to interpret, making it suitable for reporting, demonstration, and early-warning applications.

The project structure organizes all necessary scripts, data, models, and outputs in a clean hierarchy. Raw and processed data are stored in dedicated folders, trained machine learning models are saved for future predictions, and the final interactive map is stored in the maps folder. A Jupyter notebook is included to explore and visualize the cleaned data.

This system leverages several technologies, including Python 3, Pandas and NumPy for data handling, Scikit-learn for machine learning, Requests for API integration, Folium for interactive maps, and Matplotlib and Seaborn for visualization. By integrating predictive modeling with geospatial visualization, the Cloudburst Prediction System transforms numerical weather data into actionable insights, providing a comprehensive tool for situational awareness, disaster preparedness, and informed decision-making.

The project is open-source, designed for educational and research purposes, and can be extended in the future for real-time alerts, web dashboards, or integration with government disaster management systems.
